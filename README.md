## Features


- Upload art.
- Browse and get details about art.
- Buy.
- Edit art, Delete art.

  
## Tech Stack

**Client:** HTML, CSS, EJS

**Server:**  NodeJs,Express,MongoDB

  
## Installation

Clone repository


Run:
```cmd
npm install
npm start
```
